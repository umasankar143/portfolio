Electricity Price Prediction Using Machine Learning in Python

This project demonstrates a robust electricity price prediction system using various machine learning models and a dynamic dashboard.

  • Developed a dashboard using Python and the Dash library.
  • Implemented models: Linear Regression, Decision Tree, Random Forest, and Support Vector Regressor.
  • Included interactive visualizations like scatter plots, bubble plots, line graphs, pie charts, and bar graphs.
